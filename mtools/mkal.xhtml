Kalender mit Datenbank

Diese Seiten auch auf GitHub ansehen.

Der Kalender mit Datenbank, kurz MKalender, ist ein kleines Kalender-Tool mit Datenbank-Anbindung. Für das Programm wurde u.a. eine eigene Kalender-Komponente für die Lazarus ToolPalette entwickelt und mit der SQLite-Unterstützung von Lazarus experimentiert. Es wurden Feiertagsberechnungen implementiert, umfangreiche Suchmöglichkeiten eingebaut sowie eine einfache Kontakte-Tabelle.

Über ›Export‹ und ›Import‹ kann die gesamte MKalender-Datenbank als SQLDump gesichert und auch wieder geladen werden. Bei diesem Vorgang wird gleich der Index bereinigt.

Die beiden Programm-Versionen für MS Windows X und Fedora Linux basieren auf denselben Quelltexten und liegen nur als 64 Bit-Versionen vor.

Download

Die Programme werden nicht installiert, sie laufen in jedem Verzeichnis, benötigen aber dazu eine SQLite-Library. Unter Windows muss die sqlite3.dll im selben Verzeichnis stehen, wie das Programm, in der Linux-Version ist der Pfad (noch) hart codiert für Fedora. Die Programme legen außerdem Konfigurationsdateien an, in Linux im HOME unter .config/mtools/MKalender, unter WinX im User-Verzeichnis unter AppData\Roaming\mtools\MKalender. Die MKalender-Datenbank selbst kann in einem beliebigen Verzeichnis gespeichert werden.

  • mkalwinx.7z für MS Windows, 64 Bit-Version
  • mkalinx.7z für Fedora 32, 64 Bit-Version
