Find & Backup

Diese Seiten auch auf GitHub ansehen.

Find & Backup, oder kurz MFndBack, ist eine reine Studie aus der Zeit, als ich anfing, mich mit Lazarus und FreePascal zu beschäftigen. Wirklich sinnvoll ist dieses Tool nicht mehr. Aber aus dieser Studie heraus entstand dann später das Zwei-Fenster-Tool Compare & Backup.

Die beiden Programm-Versionen für MS Windows X und Fedora Linux basieren auf denselben Quelltexten und liegen nur als 64 Bit-Versionen vor. Nebenbei können mit dem Programm Requläre Ausdrücke für die Dateisuche getestet werden und es kann — allerdings nur für MS Windows X — als kleines Backup-Programm benutzt werden: Diese Funktion basiert auf den alten MS Dos Dateiattributen, die es auch unter Windows immer noch gibt. Eines dieser Attribute, faArchive, wurde von MS Dos und wird von Windows immer dann automatisch gesetzt, wenn eine Datei geändert wird. Das Programm sucht nun Dateien mit diesem gesetzten Bit, und der Nutzer kann sie auf einen anderen Datenträger sichern. Danach kann das Programm dieses Bit zurücksetzen, so dass die gesicherten Dateien beim nächsten Suchlauf nicht mehr berücksichtigt werden.

Download

Die Programme werden nicht installiert, sie sind vollständig und laufen in jedem Verzeichnis. Sie legen aber Konfigurationsdateien an, in Linux im HOME unter .config/mtools/MFndBack, unter WinX im User-Verzeichnis unter AppData\Roaming\mtools\MFndBack.

  • mfbwinx.7z für MS Windows, 64 Bit-Version
  • mfblinx.7z für Fedora 32, 64 Bit-Version
