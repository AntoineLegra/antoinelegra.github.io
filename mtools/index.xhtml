Hier können die ›4 Pieces MTools‹ als ausführbare Programme heruntergeladen werden. Sie laufen nur auf 64 Bit-Systemen mit MS Windows X einerseits, und auf 64 Bit-Systemen mit Fedora Linux andererseits. Mehrere Linux-Systeme auszuprobieren, hatte ich bisher leider keine Gelegenheit.

Auch die Quelltexte der ›4 Pieces MTools‹ werden demnächst an dieser Stelle zum Herunterladen angeboten werden.

Die Programme für MS Windows X

Die ›4 Pieces MTools‹ laufen nur auf 64 Bit-Systemen unter MS Windows X. Für den Betrieb der beiden Datenbank-Programme MKalender und MPicAlb ist die SQLite.dll im gleichen Ordner erforderlich, in dem sich die Programme befinden.

  • MFndBack
  • MCmpBack
  • MKalender
  • MPicAlb
